Position

The position property specifies the type of positioning method used ofr an element. The four position values are:

  • fixed this will pin something to a spot, and it will stay there as you scroll. (ex: nav bar) Fixed position takes elements out of the normal document flow.
  • static: this is the default setting, it is not affected by top, right, bottom, left properties. It is always positioned according to the normal flow of the page.
  • relatives: does not take the elament out of the normal document flow. still appears within ist space/div/container, but allows it to be offest a bit from the top, bottom, right, left. Top, right, bottom, and left properties will move it away from its normal position. other content will not be adjusted to fit into any gap left by the element.
  • absolute: takes element out of normal document flow. If a parent element is set to relative, and the child element is set to absolute,the child element's new (0,0) coordinate will be the upper left hand corner of the parent element instead of the whole page.
  • The z-index property allows you to override the normal stacking orderof the page. The default z-index is 0, and you can set that value to be positive or negative. An element with a greater z-index will stack in front of an element with a lower z-index. If two elements have the3 same z-index, or don't have one assigned, the element positioned last in the HTML will show on top. You must have a position assigned in oreer to assign a z-index.
