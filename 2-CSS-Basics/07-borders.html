Borders

the border property allows you to specify the style, width, and color of an element's border

Border Style

The border style property can take up to four values (top border, right border, bottom border, left border). The following border-style values are allowed:

  • dotted: dotted border
  • dashed: dashed border
  • solid: solid border
  • double: double border
  • groove: 3D groove border, depends on border-color value
  • ridge: 3D ridge border, depends on border-color value
  • inset: 3D inset border, depends on border-color value
  • none: no border
  • hidden: hidden border
  • mixed: you can set a different border style for the top, right, bottom, and left

Border Width

Very IMPORTANT: border-width will not work unless oyu have a border style set!!

Border-width anc be set to a specific size (px, em, etc) or you can use one of the following per-set settings: thin, medium, thick. The border-width property can also take up to four values (top, right, bottom, left)
